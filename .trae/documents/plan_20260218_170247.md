## 问题分析

1. **点击功能正常** - `selectRoute` 函数正确将选中的 wayline 存储到 Vuex store
2. **缺少视觉反馈** - UI 上没有显示选中状态，用户无法感知点击效果

## 解决方案

修改 `wayline.vue` 文件，添加选中状态的视觉反馈：

### 1. 添加选中状态变量
```typescript
const selectedWaylineId = ref<string>('')
```

### 2. 修改 selectRoute 函数
```typescript
function selectRoute(wayline: WaylineFile) {
  selectedWaylineId.value = wayline.id
  store.commit('SET_SELECT_WAYLINE_INFO', wayline)
}
```

### 3. 修改模板，添加绿色竖条
```html
<div class="wayline-panel" :class="{ 'selected': selectedWaylineId === wayline.id }" @click="selectRoute(wayline)">
  <div v-if="selectedWaylineId === wayline.id" class="selected-indicator"></div>
  <!-- 原有内容 -->
</div>
```

### 4. 添加 CSS 样式
```scss
.wayline-panel {
  position: relative;
  
  &.selected {
    background: #4a4a4a;
    border-left: 3px solid #52c41a;
  }
  
  .selected-indicator {
    position: absolute;
    left: 0;
    top: 0;
    bottom: 0;
    width: 3px;
    background: #52c41a;
  }
}
```

## 效果预览

- 未选中：灰色背景 (#3c3c3c)
- 选中：深灰背景 (#4a4a4a) + 左侧绿色竖条 (3px)
