## 问题分析

HBase MCP 服务器是从本地路径启动的：

* `/home/kyy/workspace/drone_simulator/mcp_servers/hbase_mcp`

MCP 配置文件中的 `hbase-mcp-server` 是错误的，应该使用本地路径。

## 解决方案

1. 修改本地 HBase MCP 服务器添加 timeout 支持
2. 更新 MCP 配置文件使用正确的启动方式

### 修改文件

`/home/kyy/workspace/drone_simulator/mcp_servers/hbase_mcp/hbase_mcp_server.py`

添加 timeout 参数：

```python
def get_connection():
    host = os.environ.get("HBASE_HOST", "118.195.150.246")
    port = int(os.environ.get("HBASE_PORT", "9090"))
    table_prefix = os.environ.get("HBASE_TABLE_PREFIX", None)
    timeout = int(os.environ.get("HBASE_TIMEOUT", "30000"))
    
    return happybase.Connection(
        host=host,
        port=port,
        table_prefix=table_prefix,
        timeout=timeout,
        autoconnect=True
    )
```
