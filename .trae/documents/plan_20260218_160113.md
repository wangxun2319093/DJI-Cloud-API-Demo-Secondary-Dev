## 问题分析

1. **HBase MCP 进程未运行** - `ps aux` 显示没有 hbase 相关进程
2. **MCP 配置错误** - 配置使用 `uvx hbase-mcp-server`，但该包在 PyPI 上不存在
3. **本地 HBase MCP 服务器可用** - 位于 `/home/kyy/workspace/drone_simulator/mcp_servers/hbase_mcp`

## 解决方案

修改 MCP 配置文件，使用本地路径启动 HBase MCP 服务器：

```json
{
  "mcpServers": {
    "HBase": {
      "command": "uvx",
      "args": ["--from", "/home/kyy/workspace/drone_simulator/mcp_servers/hbase_mcp", "hbase-mcp"],
      "env": {
        "HBASE_HOST": "118.195.150.246",
        "HBASE_PORT": "9090",
        "HBASE_TIMEOUT": "30000"
      }
    }
  }
}
```
