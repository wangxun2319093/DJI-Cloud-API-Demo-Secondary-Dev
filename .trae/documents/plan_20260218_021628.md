## 问题分析

**根本原因**：模拟器发送的 Gateway OSD 字段名格式不正确。

| 模拟器发送 (错误) | 后端期望 (正确) |
|------------------|----------------|
| `capacity_percent` | `capacityPercent` |
| `wireless_link` | `wirelessLink` |
| `dongle_number` | `dongleNumber` |
| `sdr_link_state` | `sdrLinkState` |
| `sdr_quality` | `sdrQuality` |

## 修复计划

修改 `models/gateway.py`，将字段名从下划线格式改为驼峰格式：

```python
# GatewayOsd.to_dict() 输出:
{
    "latitude": ...,
    "longitude": ...,
    "height": ...,
    "capacityPercent": 100,        # 驼峰
    "wirelessLink": {              # 驼峰
        "dongleNumber": 1,         # 驼峰
        "sdrLinkState": true,      # 驼峰
        "sdrQuality": 5,           # 驼峰
        "4g_link_state": true,     # 4g 开头的保持下划线
        "4g_quality": 5            # 4g 开头的保持下划线
    }
}
```

**注意**：`4g_` 开头的字段保持下划线格式，因为后端 `WirelessLink` 类使用 `@JsonProperty("4g_xxx")` 注解。
