## 问题分析

**根本原因**: 模拟器每次发送 OSD 数据时都会随机生成新值

```python
# device_simulator.py 第 225-229 行
dock_osd.rainfall = random.uniform(0, 2)
dock_osd.wind_speed = random.uniform(0, 5)
dock_osd.wind_direction = random.randint(0, 359)
dock_osd.temperature = 25 + random.uniform(-2, 2)
dock_osd.humidity = 50 + random.uniform(-10, 10)
```

这导致每秒数据都大幅跳动，不符合真实传感器数据的特性。

## 解决方案

修改模拟器，使数据变化更加平滑：

1. **使用渐进式变化**: 每次只在小范围内微调
2. **添加趋势模拟**: 数据有缓慢变化的趋势
3. **限制变化幅度**: 防止突变

```python
# 平滑变化示例
self.wind_speed += random.uniform(-0.1, 0.1)
self.wind_speed = max(0, min(5, self.wind_speed))

self.wind_direction += random.randint(-5, 5)
self.wind_direction = self.wind_direction % 360
```
