## 问题分析

用户要求：
1. Device 卡片高度减少 25%（从 180px 减少到约 135px）
2. 行间距增大一级（从 mt2 增大到 mt3）
3. 字体样式参考 FlightAreaItem

## FlightAreaItem 样式参考

```scss
.panel {
  height: 90px;
  font-size: 13px;
  .title {
    height: 30px;
    font-weight: bold;
    margin: 0px 10px;
  }
}
```

## 优化方案

### Online Devices（设备卡片）

| 属性 | 当前值 | 优化后 |
|------|--------|--------|
| 卡片高度 | 180px | 135px |
| 行间距 | mt2 | mt3 |
| 行高 | 16px | 14px |
| 字体 | 12px | 13px |

### Dock（机场卡片）

| 属性 | 当前值 | 优化后 |
|------|--------|--------|
| 卡片高度 | 120px | 90px（参考 FlightAreaItem） |
| 行间距 | mt2 | mt3 |
| 行高 | 16px | 14px |
| 字体 | 12px | 13px |
