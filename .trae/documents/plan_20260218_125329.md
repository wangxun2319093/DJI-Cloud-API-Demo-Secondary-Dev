## 问题分析

**根本原因**：第90行的 `flex-align-center` 类导致内容垂直居中，而不是从顶部开始排列。

**当前代码**：
```html
<div style="border-radius: 2px; height: 100%; width: 100%;" class="flex-row flex-justify-between flex-align-center">
```

## 解决方案

移除 `flex-align-center`，让内容从顶部开始排列：

```html
<div style="border-radius: 2px; height: 100%; width: 100%;" class="flex-row flex-justify-between">
```

这样状态数据会从顶部开始排列，而不是居中显示。
