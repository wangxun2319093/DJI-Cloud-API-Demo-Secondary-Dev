## 修复方案

### 问题原因
模拟器发送的数据格式不符合大疆上云 API 文档。遥控器系统需要分别发送两种 OSD 消息：
1. **遥控器 OSD** - 只包含遥控器自身状态
2. **无人机 OSD** - 需要通过子设备 SN 发送

### 修复计划

#### 1. 在数据库中为 GATEWAYSIM001 添加子设备（无人机）
```sql
-- 添加无人机设备
INSERT INTO manage_device (device_sn, device_name, nickname, workspace_id, device_type, sub_type, domain, parent_sn)
VALUES ('DRONE_SIM001', 'DJI M30T', 'Sim Drone', 'e3dea0f5-37f2-4d79-ae58-490af3228069', 89, 0, 1, 'GATEWAYSIM001');

-- 更新 Gateway 的 child_sn
UPDATE manage_device SET child_sn = 'DRONE_SIM001' WHERE device_sn = 'GATEWAYSIM001';
```

#### 2. 更新模拟器
修改模拟器分别发送遥控器 OSD 和无人机 OSD：
- `thing/product/GATEWAYSIM001/osd` - 遥控器数据
- `thing/product/DRONE_SIM001/osd` - 无人机数据（包含速度、距离、风速、GPS等）

#### 3. 更新前端类型定义
确保 `GatewayOsd` 类型只包含遥控器字段，无人机数据从 `deviceInfo[device.sn]` 获取
