## HBase MCP 配置方案

### 问题分析

HBase MCP Server 使用环境变量配置连接：

* `HBASE_HOST`: 默认 localhost

* `HBASE_PORT`: 默认 9090

当前连接失败是因为默认连接 localhost:9090，需要修改为 118.195.150.246:9090。

### 解决方案

修改 MCP 配置文件 `/home/kyy/.config/claude/claude_desktop_config.json`，添加 HBase MCP Server 配置：

```json
{
  "mcpServers": {
    "s3-minio": {
      "command": "uvx",
      "args": ["s3-mcp"],
      "env": {
        "AWS_ACCESS_KEY_ID": "minioadmin",
        "AWS_SECRET_ACCESS_KEY": "minioadmin123",
        "AWS_ENDPOINT_URL": "https://minio.cragc.com.cn"
      }
    },
    "HBase": {
      "command": "uvx",
      "args": ["hbase-mcp-server"],
      "env": {
        "HBASE_HOST": "118.195.150.246",
        "HBASE_PORT": "9090"
      }
    }
  }
}
```
